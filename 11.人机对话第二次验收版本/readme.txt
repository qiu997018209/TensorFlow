对话系统第二版改进点:
	1.采用flask框架，web服务更稳定
	2.满足陈勇要求:每次启动服务前不训练，根据json命令才训练，0条数据服务可启动
	3.满足陈总要求:一份代码对外提供所有服务，无需拷贝多个文件
	4.所有功能均支持命令行参数配置
	5.优化剩余时间计算方式，避免之前数据量太多时，时间没有更新，测试人员以为训练没有开始
	6.优化代码结构，节省内存，调参和对外服务均为一份代码，提高可维护性
	7.训练阶段可对话
	
使用方法:
	cd server
	python3 run_server.py -u 2 -m ../data/runs/test1 -p 1111

测试方法:
	cd server
	python3 http_test.py
	输入:l 代表查询训练进度
	输入:t 代表重新训练
	输入:c 代表对话

bug修改记录:
        1.重新训练的时候数据可能变化，需要重新载入mysql数据
        2.重新训练完成进度条100%，但是模型载入期间用户问问题打不上，修改为将模型载入后才将进度条设置为100%
		3.第一次启动没有数据，加入一条数据后重新训练，此时self.args.vocab_size为0，导致build_vocab_size报错
